# Royco Dusk

Technical specification for Royco Dusk.

The compiled paper is available at [royco_dusk_paper.pdf](./royco_dusk_paper.pdf).

## Building

The paper is written in LaTeX and built with `latexmk`.

| Command | Description |
| --- | --- |
| `make` | Build the paper to `./royco_dusk_paper.pdf`. |
| `make watch` | Rebuild automatically on file changes. |
| `make clean` | Remove build artifacts. |
